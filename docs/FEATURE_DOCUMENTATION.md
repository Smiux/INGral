# MyMathWiki 功能文档

## 1. 项目概述

MyMathWiki 是一个基于 React 和 Supabase 的知识图谱应用，旨在提供一个社区驱动的数学知识平台，支持创建、分享和可视化数学概念。

### 技术栈
- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **后端服务**: Supabase
- **图形可视化**: D3.js
- **Markdown 处理**: MarkdownIt
- **数学公式渲染**: KaTeX
- **路由管理**: React Router
- **样式方案**: Tailwind CSS

## 2. 核心功能

### 2.1 文章管理
- **创建文章**: 支持匿名提交和编辑
- **文章版本控制**: 完整的版本历史记录和还原功能
- **文章链接**: 自动提取和管理文章间的引用关系
- **标签系统**: 支持文章标签管理和分类
- **阅读计数**: 自动统计文章阅读量

### 2.2 知识图谱
- **图谱创建与编辑**: 可视化创建和管理知识图谱
- **图谱模板**: 支持使用和创建图谱模板
- **图谱导出**: 支持 JSON、GraphML 和 CSV 格式导出
- **图谱导入**: 支持从文件导入图谱数据
- **节点和连接管理**: 灵活的节点和连接编辑功能

### 2.3 搜索功能
- **全文搜索**: 基于 PostgreSQL tsvector 的高效全文搜索
- **标签过滤**: 支持按标签筛选搜索结果
- **搜索建议**: 智能搜索建议功能
- **搜索缓存**: 优化搜索性能的缓存机制

### 2.4 评论系统
- **匿名评论**: 支持匿名用户发表评论
- **评论嵌套**: 支持评论回复和嵌套结构
- **评论投票**: 支持评论点赞和点踩

### 2.5 数据分析
- **页面访问统计**: 跟踪页面访问量和用户行为
- **热门内容分析**: 识别热门文章和图谱
- **用户活跃度**: 分析用户参与度和活跃度
- **流量来源**: 跟踪访问来源渠道

### 2.6 导出功能
- **文章导出**: 支持 Markdown、HTML 和 PDF 格式
- **图谱导出**: 支持 JSON、GraphML 和 CSV 格式
- **批量导出**: 支持多文件打包导出

### 2.7 内容审核
- **审核机制**: 完整的文章审核流程
- **审核状态**: 支持 pending、approved、rejected、needs_revision 四种状态
- **准确性评分**: 0-5分的准确性评分系统
- **审核历史**: 完整的审核日志记录
- **验证机制**: 支持文章内容验证和标记

### 2.8 数学计算
- **SymPy 集成**: 支持在文章中嵌入 SymPy 计算单元格
- **实时计算**: 支持数学表达式的实时计算和结果显示
- **计算历史**: 保存计算历史，支持查看和重用
- **多种计算类型**: 支持代数、微积分、线性代数等多种计算类型

### 2.9 3D 可视化
- **Three.js 集成**: 基于 Three.js 的 3D 图形渲染
- **3D 模型导出**: 支持导出 3D 模型为常见格式
- **交互式控制**: 支持旋转、缩放、平移等交互式操作
- **多种图表类型**: 支持散点图、曲面图、线图等多种 3D 图表类型

### 2.10 数据库监控
- **表结构检查**: 验证数据库表完整性
- **健康检查**: 实时监控数据库连接状态
- **初始化支持**: 数据库初始化和迁移支持
