# 知识图应用

一个基于React、TypeScript和Vite构建的知识图应用，用于创建、管理和可视化知识文章与图表。

## 功能介绍

### 📝 文章管理
- **富文本编辑器**: 支持Markdown语法的文章编辑器，支持插入图片、视频和数学公式
- **文章查看器**: 优雅的文章阅读体验，支持代码高亮和数学公式渲染
- **版本历史**: 记录文章的编辑历史，支持回滚到之前的版本
- **评论系统**: 支持文章评论和回复，支持点赞/点踩功能
- **标签管理**: 支持为文章添加标签，便于分类和搜索
- **可见性控制**: 支持公开和未列出两种可见性设置

### 📊 数据可视化
- **多种图表类型**: 支持柱状图、折线图、饼图、热力图和Sankey图
- **交互式图表**: 支持图表悬停、点击事件和动态数据更新
- **自定义配置**: 支持自定义图表颜色、样式和配置
- **响应式设计**: 图表会根据容器大小自动调整

### 🔗 知识图
- **2D可视化**: 基于D3.js的2D知识图可视化
- **3D可视化**: 基于Three.js的3D知识图可视化
- **节点管理**: 支持添加、编辑和删除节点
- **关系管理**: 支持定义和可视化节点之间的关系
- **图分析**: 支持图结构分析和关系查询
- **交互式操作**: 支持拖拽、缩放和节点选择

### 🔍 智能搜索
- **全文搜索**: 支持文章内容的全文搜索
- **标签搜索**: 支持按标签搜索文章
- **语义搜索**: 基于语义的智能搜索功能
- **搜索过滤**: 支持按发布日期、作者等条件过滤搜索结果

### ⌨️ 键盘快捷键
- **编辑器快捷键**: 支持常用的编辑快捷键
- **全局快捷键**: 支持应用级别的键盘导航
- **自定义快捷键**: 支持自定义快捷键配置

### 🎨 主题支持
- **明暗主题**: 支持浅色和深色主题切换
- **主题定制**: 支持自定义主题颜色和样式
- **自动切换**: 支持根据系统设置自动切换主题
